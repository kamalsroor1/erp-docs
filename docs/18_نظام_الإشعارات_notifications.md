# 18 نظام الإشعارات (Notifications System)
**مشروع Ebraa ERP - التصميم التقني للإشعارات والمنطق اللحظي**

---

## 1. أنواع الإشعارات (Notification Types)

يعتمد النظام على ثلاثة مسارات للإشعارات لضمان وصول المعلومة للتاجر في الوقت المناسب:

### أ. الإشعارات اللحظية (Real-time - Laravel Reverb)
- **الغرض:** تنبيه الموظف أو المدير أثناء فتح التطبيق.
- **الحالات:**
    - وصول منتج لنقطة إعادة الطلب (بينما البائع يبيع).
    - نجاح عملية تحويل مخزني واردة من فرع آخر.
    - دفع قسط من قبل عميل (تنبيه للمدير).

### ب. إشعارات تليجرام (Telegram Bot API)
- **الغرض:** تنبيه صاحب المحل خارج التطبيق وإرسال التقارير.
- **الحالات:**
    - ملخص المبيعات اليومي (عند تصفية اليومية).
    - تنبيهات محاولات الدخول الفاشلة أو العمليات الحساسة (مثل البيع بأقل من سعر التكلفة).
    - إرسال نسخة PDF من الفاتورة للعميل.

### ج. إشعارات النظام (Database Notifications)
- **الغرض:** سجل تاريخي للإشعارات يمكن الرجوع إليه.
- **الحالات:** كل ما سبق يتم أرشفته في جدول `notifications`.

---

## 2. هيكل البيانات (Database Schema)

```sql
CREATE TABLE notifications (
    id              UUID PRIMARY KEY,
    type            VARCHAR(255),                    -- اسم فئة الإشعار (StockAlert, NewInstallment)
    notifiable_type VARCHAR(255),                    -- App\Models\User
    notifiable_id   BIGINT,
    data            JSONB,                           -- محتوى الإشعار (العنوان، الرسالة، الرابط)
    read_at         TIMESTAMP NULL,
    created_at      TIMESTAMP DEFAULT NOW()
);
```

---

## 3. منطق التنبيهات المجدولة (Scheduled Alerts)

تُستخدم الـ **Laravel Task Scheduling** للتحقق من المهام التي تحتاج تنبيهات استباقية:

### أ. تنبيهات الأقساط والشيكات
- يتم تشغيل Job يومياً في تمام الساعة 8 صباحاً.
- يقوم بالبحث في جدول `installments` و `cheques` عن المواعيد التي تستحق اليوم أو غداً.
- يرسل إشعار تليجرام للمدير بقائمة الديون المستحقة التحصيل.

### ب. تنبيهات انتهاء الضمان
- يتم التحقق أسبوعياً من الأجهزة التي سينتهي ضمانها خلال 7 أيام.
- يُرسل تقرير للمحل ليتمكن من تقديم "عرض صيانة وقائية" للعميل (رفع نسبة الولاء).

---

## 4. التكامل التقني (Technical Integration)

1. **Telegram Bot:**
    - يتم تخزين `telegram_chat_id` في جدول `users`.
    - استخدام مكتبة `irazasyed/telegram-bot-sdk` لإرسال الرسائل والملفات.

2. **Laravel Reverb:**
    - إعداد WebSocket server على السيرفر.
    - استخدام `Echo.private('branch.' + branchId)` لبث الإشعارات لموظفي فرع محدد فقط.

---

> [!TIP]
> يُنصح في المرحلة الثانية إضافة **SMS API** (مثل SMS Misr) كبديل لمن لا يفضل استخدام تليجرام، مع العلم أن تليجرام هو الخيار الافتراضي المجاني للمنصة.
