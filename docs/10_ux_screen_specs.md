# 10 مواصفات الشاشات (Screen Specifications)
**مشروع Ebraa ERP - تفاصيل واجهة المستخدم**

---

## 1. شاشة البيع السريع (POS Screen)
*   **النوع:** PWA / Desktop Web
*   **المكونات:**
    - حقل بحث ذكي (يدعم الباركود والسيريال).
    - قائمة السلة (تحديث لحظي عبر Inertia v3 Optimistic Updates).
    - أزرار اختصار للمبالغ النقدية (50, 100, 200).
    - زر "إتمام وطباعة".
    - خيار "إرسال نسخة لـ **تليجرام**".

---

## 2. لوحة تحكم صاحب المحل (Owner Dashboard)
*   **النوع:** Responsive Web (Mobile Optimized)
*   **المكونات:**
    - بطاقات إحصائية (إجمالي المبيعات، الأرباح، الديون).
    - رسم بياني لحركة المبيعات خلال الأسبوع.
    - قائمة "آخر العمليات" مع زر التفاصيل.
    - قسم "التنبيهات العاجلة" (مخزون منخفض، مواعد سداد ديون).

---

## 3. شاشة الجرد بالموبايل (Inventory Mobile Screen)
*   **النوع:** PWA (Mobile Only)
*   **المكونات:**
    - نافذة الكاميرا (تستخدم WebAssembly للمسح السريع).
    - عداد "القطع الممسوحة".
    - قائمة "آخر القطع التي تم مسحها" مع وصفها.
    - زر "مطابقة نهائية".

---

## 4. بوابة المسوقين (Affiliate Portal)
*   **النوع:** Web
*   **المكونات:**
    - رابط الإحالة الفريد (مع زر نسخ).
    - إحصائيات: عدد الضغطات، عدد المسجلين، عدد المشتركين الفعليين.
    - رصيد العمولات (معلق، متاح للسحب).
    - سجل عمليات السحب.
    - زر ربط حساب **تليجرام** لاستقبال إشعارات العمولات.
