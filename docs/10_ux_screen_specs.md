# 10 مواصفات الشاشات (Screen Specifications)
**مشروع Ebraa ERP - تفاصيل واجهة المستخدم**

---

## 1. شاشة البيع السريع (POS Screen)
*   **النوع:** PWA / Desktop Web
*   **المكونات:**
    - حقل بحث ذكي (يدعم الباركود، السيريال الفريد، الماركة، والموديل).
    - قائمة السلة (تحديث لحظي عبر Inertia v3 Optimistic Updates).
    - عرض حالة الضمان لكل صنف في السلة آلياً.
    - أزرار اختصار للمبالغ النقدية (50, 100, 200).
    - زر "إتمام وطباعة" (يدعم الطابعات الحرارية و A4).
    - خيار "إرسال نسخة لـ **تليجرام**" (للبائع والمشتري).

---

## 2. لوحة تحكم صاحب المحل (Owner Dashboard)
*   **النوع:** Responsive Web (Mobile Optimized)
*   **التصميم:** يعتمد على **"البلاطات" (Tiles)** للوصول السريع للعمليات المتكررة.
*   **لوحة الوصول السريع (Quick Access Tiles):**
    - **العمليات:** [بيع، شراء، مرتجع بيع، مرتجع شراء، عرض أسعار].
    - **المخازن:** [جرد مخزن، تسوية مخزن، تحويل لمخزن].
    - **الماليات:** [صرف (مدفوعات)، قبض (مقبوضات)].
    - **الإعدادات:** [إعداد النظام، إدخال الأصناف، إدخال الحسابات].
    - **التقارير:** [تحليل المبيعات، الحركة اليومية].
*   **المكونات الإحصائية:**
    - بطاقات إحصائية (إجمالي المبيعات، الأرباح، الديون) - تحديث لحظي عبر Reverb.
    - رسم بياني لحركة المبيعات والمنتجات الأكثر طلباً.

---

## 3. شاشة الجرد بالموبايل (Inventory Mobile Screen)
*   **النوع:** PWA (Mobile Only)
*   **المكونات:**
    - نافذة الكاميرا (تستخدم WebAssembly للمسح السريع).
    - عداد "القطع الممسوحة".
    - قائمة "آخر القطع التي تم مسحها" مع وصفها.
    - زر "مطابقة نهائية".

---

## 4. بوابة المسوقين (Affiliate Portal)
*   **النوع:** Web
*   **المكونات:**
    - رابط الإحالة الفريد (مع زر نسخ).
    - إحصائيات: عدد الضغطات، عدد المسجلين، عدد المشتركين الفعليين.
    - رصيد العمولات (معلق، متاح للسحب).
    - سجل عمليات السحب.
    - زر ربط حساب **تليجرام** لاستقبال إشعارات العمولات.

---

## 5. هيكل القائمة العلوية (Navbar Structure)
تعتمد الواجهة على قائمة علوية مبوبة للوصول الشامل لجميع الموديولات:
- **النظام:** (بيانات الشركة، الفروع، المستخدمين).
- **البضاعة:** (الأصناف، المخازن، تحويلات، جرد).
- **الحسابات:** (العملاء، الموردين، كشوفات الحساب).
- **الخزينة:** (حركة الخزينة، التحويلات المالية، البنوك).
- **الفواتير:** (مبيعات، مشتريات، مرتجعات).
- **تقارير:** (تقارير مالية، تقارير مخزنية، أرباح).
- **تقسيط / مساعدة / خدمة العملاء.**

---

## 6. شاشات الخزينة والماليات (Financial Screens)

### أ. حركة الخزينة (Treasury Movement)
- **المؤشر البصري:** عرض "رصيد الخزينة الحالي" بخط كبير وواضح (باللون الأخضر).
- **الفلاتر:** اختيار الخزينة/البنك، تحديد التاريخ، البحث بالوصف.
- **جدول الحركات:** (رقم، وصف، مرجع، تاريخ ووقت، وارد، صادر، رصيد تراكمي لحظي).

### ب. كشف الحساب التفصيلي (Statement of Account)
- **المحتوى:** تجميع كافة الحركات (فواتير، سندات، تسويات) في جدول واحد.
- **التوازن:** توضيح الرصيد الافتتاحي، الرصيد الختامي، والرصيد التراكمي بعد كل حركة.

---

## 7. شاشة إدارة الصلاحيات (RBAC Settings)
- **التصميم:** مقسمة إلى تبويبات (Tabs): [صلاحيات عامة، مستندات يومية، البضاعة، الحسابات، الخزينة].
- **مصفوفة CRUD:** لكل شاشة (مثل مبيعات) يوجد 4 خيارات (Checkboxes): [إستعراض، إضافة، تعديل، حذف].
- **التحكم الخاص:** قسم لأعلام الميزات (Feature Flags) مثل (تعديل السعر، البيع بأقل من التكلفة، إظهار الربحية).
