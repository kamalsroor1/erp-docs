## 1. استراتيجية الاختبار الثلاثية (Testing Pyramid)

```
         /\
        /E2E\        ← 10% (Browser Tests - الأبطأ والأغلى)
       /──────\
      / Feature\     ← 30% (API Tests - اختبار كل Endpoint)
     /──────────\
    /  Unit Tests \  ← 60% (الوحدات المنطقية - الأسرع والأكثر)
   /______________\
```

- **Unit Tests** (PestPHP): العمليات الحسابية، Business Rules، التحقق من الصحة.
- **Feature Tests** (PestPHP + Laravel HTTP Testing): كل API Endpoint.
- **Browser Tests** (Playwright): تدفقات المستخدم الحرجة (البيع، الجرد).

## 3. معايير القبول العامة (Global Acceptance Criteria)
تُعتبر كل Feature "جاهزة للإنتاج" إذا:
1. ✅ تغطية اختبارات الوحدة لا تقل عن **80%**.
2. ✅ Response Time لكل API لا يتجاوز **500ms** في الأحوال العادية.
3. ✅ اجتياز **جميع** اختبارات عزل البيانات (TI-01 → TI-05).
4. ✅ لا توجد أخطاء JavaScript في الـ Console على Chrome وFirefox.
5. ✅ الواجهة تعمل على شاشات 320px (موبايل صغير) وحتى 1920px.

## 5. أمثلة اختبارات PestPHP جاهزة

### اختبار عزل البيانات (الأهم)
```php
test('tenant cannot access another tenant products', function () {
    $tenantA = Tenant::factory()->withOwner()->create();
    $tenantB = Tenant::factory()->withOwner()->create();
    $product = Product::factory()->create(['tenant_id' => $tenantA->id]);

    $this->actingAs($tenantB->owner)
         ->getJson("/api/products/{$product->id}")
         ->assertStatus(403);
});
```

### اختبار منع بيع سيريال مباع
```php
test('cannot sell an already sold serial number', function () {
    $tenant = Tenant::factory()->withSalesman()->create();
    $item = DeviceItem::factory()->sold()->create(['tenant_id' => $tenant->id]);

    $this->actingAs($tenant->salesman)
         ->postJson('/api/sales', [
             'items' => [['device_item_id' => $item->id]]
         ])
         ->assertStatus(422)
         ->assertJsonPath('message', 'هذه القطعة تم بيعها بالفعل');
});
```

### اختبار صلاحيات الأدوار
```php
test('salesman cannot modify product price', function () {
    $tenant = Tenant::factory()->withSalesman()->create();

    $this->actingAs($tenant->salesman)
         ->patchJson("/api/products/{$tenant->products->first()->id}", [
             'sell_price' => 999
         ])
         ->assertStatus(403);
});
```

### اختبار دقة حساب الفاتورة
```php
test('invoice total calculation is accurate', function () {
    // سعر 1999.99 × 2 قطعة = 3999.98 - خصم 10% = 3599.982 → 3599.98
    $sale = createSaleWith(price: 1999.99, quantity: 2, discount: 10);
    expect($sale->total)->toBe(3599.98);
});
```
